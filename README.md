# VeriTix Web

VeriTix is a **blockchain-powered ticketing platform** designed to provide secure, efficient, and transparent event ticketing. By leveraging the **Stellar blockchain ecosystem**, VeriTix ensures tamper-resistant credentialing and settlement of ticket-related data, enabling a seamless user experience for both attendees and organizers.

VeriTix offers:

- **Configurable organizer controls** for pricing, transfer limits, and revenue sharing  
- **Mobile-friendly workflows** for purchase and event entry  
- **Transparent transaction logs** anchored on Stellar for auditability  

Unlike traditional centralized ticketing systems, VeriTix uses distributed ledger technology to increase trust, reduce costs, and protect the rights of all users.
